## 🚀 Featured Research & Projects

### 🧠 Neuroimaging & BCI (Brain-Computer Interface)
* **[Cross-Subject MEG Decoding](https://github.com/sathya8998/Cross-Subject-MEG-Imagery-Decoding-Using-Riemannian-Deep-Learning):** Utilizing **Riemannian Geometry** and Tangent Space mapping to decode imagined objects from MEG signals.
* **[Stroke Recovery Transformer](https://github.com/sathya8998/Multimodal-EEG-fNIRS-Fusion-Using-Transformers-for-Motor-Decoding-in-Stroke-Patients):** A Spatio-Temporal Transformer fusing **EEG & fNIRS** for zero-calibration motor intent decoding.
* **[Self-Supervised Brain MRI](https://github.com/sathya8998/Self-Supervised-3D-Multimodal-MRI-Classification):** Using **3D SimCLR** and ResNet to classify cognitive expertise (Chess Grandmasters) from sMRI/fMRI volumes.
* **[Cognitive Load Attention](https://github.com/sathya8998/EEG-fNIRS-Eye-Tracking-Cognitive-Load-Attention-Network):** Multi-modal late fusion for mental strain estimation using attention-based sensor weighting.

### 🏥 Clinical Computer Vision & Genomics
* **[DermAI Safety Routing](https://github.com/sathya8998/DermAI-Domain-Aware-Soft-Routing-for-Skin-Classification):** A "Safety-First" skin lesion classifier with **Abstention Protocols** and Domain-Aware routing.
* **[Stroke Detection & Segmentation](https://github.com/sathya8998/Brain-Stroke-Detection-and-Segmentation-using-Multi-Task-ResNet-UNet):** Multi-task **ResNet-UNet** for simultaneous pathology diagnosis and lesion localization in CT scans.
* **[Gait Scoring (CVPR 2026)](https://github.com/sathya8998/Spatio-Temporal-Skeleton-Fusion-for-Clinical-Gait-Scoring):** Biomechanical **ST-GCN** for fine-grained gait analysis in children with Cerebral Palsy.
* **[Genomic Biomarker Discovery](https://github.com/sathya8998/Hybrid-Graph-Attention-Network-for-Ischemic-Stroke-Detection-and-Biomarker-Discovery):** Using **Graph Attention Networks (GAT)** and GNNExplainer to identify stroke-related genetic drivers.

### 🤖 Robotics & Edge Sensing
* **[Terrain-Aware Navigation](https://github.com/sathya8998/Multi-Modal-Fusion-and-Multi-Task-Learning-for-Terrain-Aware-Navigation):** Unified MTL framework fusing **RGB, PointNet (LiDAR), and LSTM (IMU)** for autonomous robot perception.
* **[Urban Environment Prediction](https://github.com/sathya8998/Multi-Modal-IMU-GNSS-Fusion-for-Indoor-Outdoor-Environment-Classification-with-XAI):** XGBoost-based sensor fusion for Indoor/Outdoor context detection with **SHAP** explainability.
